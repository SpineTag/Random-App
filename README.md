# Grand Strategy Prototype

A lightweight browser-based grand strategy game prototype.

## How to play

1. Open `index.html` in your browser.
2. Select a region to view its details.
3. Use actions to recruit, fortify, move troops, or attack nearby enemy regions.
4. Click `End Turn` to collect income and let AI empires act.
5. Capture 8 regions to win, or lose if enemies control 8 regions.

## Files

- `index.html` — game layout and controls
- `style.css` — UI styling
- `game.js` — game logic and AI

Enjoy building and expanding your realm!
